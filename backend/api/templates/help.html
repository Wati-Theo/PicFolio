API Documentation

/api/upload

Endpoint to upload photos/videos

Method: POST
Parameters:
  • username (string) - User's name
  • asset (file) - Photo or video to upload
Response: JSON - {'message': 'Uploaded successfully'} or {'error': 'Invalid file type'}

/api/delete/<photo_id>

Endpoint to delete a photo

Method: DELETE
Parameters:
  • photo_id (int) - ID of the photo to delete
Response: JSON - {'message': 'Photo deleted successfully'} or {'error': 'Photo not found'}

/api/preview/<photo_id>

Endpoint to get a photo/video preview

Method: GET
Parameters:
  • photo_id (int) - ID of the photo/video
Response: Image or video preview

/api/asset/<photo_id>

Endpoint to get a photo/video original

Method: GET
Parameters:
  • photo_id (int) - ID of the photo/video
Response: Original photo or video

/api/list

Endpoint to get a list of photos/videos

Method: GET
Response: JSON - [{"id": 1, "name": "Photo 1", "date": "2023-10-26"}, {"id": 2, "name": "Photo 2", "date": "2023-10-27"}, {"id": 3, "name": "Photo 3", "date": "2023-10-28"}]
